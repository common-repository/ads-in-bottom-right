== Description ==

This plugin allow you show advertising in bottom right, you can put images, flash or any html code which you want. I've activated this plugin on http://blog.casanova.vn and you can see it here. If you have any comment, please give it on my blog. I'll reply you.

== Installation ==

1. Download the plugin.
2. Upload it to the plugins folder of your blog.
3. Goto the Settings --> Ads in bottom right to config your Ads.
